Implemented Pages

User Profile Page

  • Profile Section
    • HTML:
      • Used "profile-container" to group the user's profile picture, name, and bio.
      • Added an input field for users to upload a profile picture.
      • Included a textarea for users to write a short bio.
    • CSS:
      • display: flex; flex-direction: column; align-items: center; ensures a clean and centered layout.
      • border-radius: 50%; makes the profile picture circular.
      • box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.2); adds depth to the profile section.
  • Preferences Section
    • HTML:
      • Used "preferences-container" to group user preferences like dietary choices and notifications.
      • Added checkbox inputs for users to select their preferences.
    • CSS:
      • display: grid; grid-template-columns: repeat(2, 1fr); organizes preferences into a grid layout.
      • gap: 20px; ensures proper spacing between elements.

Site Map Page

  • Site Map Structure
    • HTML:
      • Used "sitemap-container" to organize all links hierarchically.
      • Added ul and li elements to create a nested list structure.
    • CSS:
      • list-style-type: none; removes default bullet points for a cleaner look.
      • padding-left: 20px; creates indentation for nested lists.
      • color: #6b4f1d; matches the theme's color scheme.
  • Interactive Elements
    • HTML:
      • Added a tags with href attributes for navigation.
      • Included hover effects for better interactivity.
    • CSS:
      • transition: color 0.3s ease; creates smooth hover effects.
      • font-weight: bold; highlights active links.

Content Page (Sustainable Farming)

  • Title & Structure
    • HTML:
      • Used <h1 class="article-heading"> for the main title.
      • Wrapped content in <main class="article-container"> for a structured layout.
    • CSS:
      • font-size: 32px; ensures prominence.
      • text-align: center; centers the title.
      • color: #6b4f1d; matches the theme.
  • Paragraphs & Lists
    • HTML:
      • Used <p> for paragraphs and <ul> for lists.
      • Added <blockquote> for key quotes.
    • CSS:
      • text-align: justify; ensures clean alignment.
      • color: #444; provides soft contrast for readability.
      • font-size: 18px; ensures readability.
  • Images & Media
    • HTML:
      • Used <div class="article-image"> to wrap images.
      • Added alt attributes for accessibility.
    • CSS:
      • max-width: 100%; ensures responsiveness.
      • border-radius: 10px; smoothens edges.
      • box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.2); adds depth.

Challenges and Lessons Learned

  • User Profile Page

    I found it hard to make the profile page look good on all devices. I used CSS to make it responsive.

    • Learned how to use flexbox for layout.
    • Used CSS properties like `font-smoothing` and `letter-spacing` for better readability.
  • Site Map Page

    It was tricky to organize all the links in a clear way. I used lists and CSS to style them.

    • Learned to use unordered lists for organizing links in the site map.
  • Sustainable Farming Page

    I struggled to make the farming page look interesting. I added images and styled the text.

    • Learned to use CSS for text alignment.
